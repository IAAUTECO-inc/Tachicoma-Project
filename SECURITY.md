# Security Policy

## Supported Versions
Only the latest hardened release of Tachikoma (v1.x) is supported for security updates.

## Reporting a Vulnerability
As this framework is used in critical assistive robotics, security is our absolute priority. Please report any vulnerability via encrypted mail to: **infrastructure@ia-aut-eco.fr**.

**Do not open a public issue for security vulnerabilities.** We follow a coordinated disclosure protocol to ensure user safety.
